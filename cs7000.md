The [Connect Systems CS7000 M17 Plus](https://www.connectsystems.com/amateur-products/CS7000-M17-plus.html) is a radio that supports DMR and M17. The [CS7000 M17 PLUS CPS](https://www.connectsystems.com/amateur-software/CS7000_m17_plus_software.html) allows you to import digital contacts, channels, and zones from Excel spreadsheets. I wrote an application to convert a codeplug from the Anytone 878UVII Plus radio (CPS 3.04 through 4.0) into the spreadsheets that can be imported into the CS7000 CPS. The application is available as a stand-alone executable and is compatible with 64-bit Windows 10 and Windows 11 operating systems. Windows 7 is not supported. 

## Features
- Supports codeplugs from Anytone 878UVII Plus (CPS 3.04 through 4.0)
- Provides option to load stock channels and zones that comes pre-loaded in the CS7000 firmware (9.00.93)
- Provides option to load Contacts into the CS7000 using either the Table or Direct method. See Note 1
- Checks for duplicate Contacts and Channels and asks user to select which name to use. Codeplug updated accordingly. See user guide for details.
- Removes any VHF channels from import as the CS7000 is a UHF only radio. In event a Zone only has VHF channels it is not imported.

Note 1: Consult Connect systems [CS7000 M17 Plus User Guide](https://www.connectsystems.com/products/top/radios/CS7000_M17/Using%20the%20CS7000%20M17%20Radio.pdf) user guide for more about the Table and Direct method. The documentation is contained in the ZIP file for the firmware for the CS7000.

## Limitations
- Scan Lists are not imported. CS7000 CPS does not provide a way to import scan lists
- Import of fields related to Squelch method needs work.  Guidance needed from users and Connect systems on how this should be working.
