## Task 1

Winlink Image Xfer:
a. Find "Cumulonimbus cloud seen from 38,000 feet" (cumulonimbus-cloud1.jpeg) image at https://www.noaa.gov/jetstream > Clouds and download.
b.The Winlink attachment Image 'Resizing Tool' should be used. Suggested procedures:  Create 'Message' > 'New Message' > select 'Attachments' > 'Add' your selected image > 'Edit/Resize-Image' > 'Change Size' > make your changes > 'Save? > 'Save Changes' > 'Finished' > Address and post to your Outbox. Send your messages via Target Stations for relay, or directly to N3XL.
c. When relaying to N3XL via a Target Station, place IMAGE XFER - RELAY TO N3XL in the Subject line. (Well defined subject lines are needed to achieve meaningful Winlink 309 message logs. Develop good habits now.)
d. RMS station WA3YOO-9 will be configured to serve as a Post Office during internet outages. Therefore, it can be used to send directly to N3XL in the Challenge. In a real internet outage, N3XL would pick up your message at the WA3YOO-9 Post Office, but in this exercise your email will be delivered via the internet.

TASK2: VarAC Image Xfer: 
a. Find the "Airflow around highs and lows" (airflow.png) image at https://www.noaa.gov/jetstream > Synoptic Meteorology > Wind (Origin of Wind) and download.
a. If desired, set up an outgoing files directory in the QSO tab of your Settings, C:\VarAC\Outgoing_Files is suggested.
b.The VarAC 'Image Shrinker' tool should be used. Suggested procedures: go to 'Tools? > Image Shrinker > ?LOAD IMAGE' > resize the image using the 'Resolution' and 'Quality' sliders > check the 'My callsign' box > ?SAVE TO DISK?. Note the file location that pops up upon saving to disk. 
c. Once connected to your Target Station, use the 'SEND FILE' button, select the file from the file location. 
d. If relaying via a Target Station, type RELAY TO N3XL in the 'New message' window at the bottom of the main window to inform the Target Station and then SEND.

## Task 3

Fldigi HF (mode MFSK32) Image Xfer: 
a. Review the image transfer training at https://drive.google.com/file/d/11WT6GyQkOqavS0-ejTaqqcBO4KkEoJOG/view?usp=sharing. 
b. Establish a connection with your Target Station, see ESTABLISHING FLDIGI CONNECTIONS below.
c. In blue transmit area of the screen, right click and select 'Send Image' Press the 'Load' button in the dialog window and select a weather-related image from https://www.noaa.gov/jetstream for transmit. The file selection dialog has a preview capability so you will see what the image looks like. 
d. Send your picture via a Target Station for relay, or directly to N3XL. After sending secure the Tx/Rx function.
e. If the transmission fails using X1, be prepared to resize the image using the X2  button and/or select XmtGry in the dialog window. This will shrink the image size at the receiving end.
f. If relaying via a Target Station, type RELAY TO N3XL in the SEND window, before or after sending the image.

* ESTABLISHING FLDIGI CONNECTIONS (P = Participant. TS = Target Station)
P - Tune radio to TS dial frequency. Use QSY button to adjust frequency to the 1500 sweet spot.
TS - QST QST de WX3F WX3F, QST QST de WX3F WX3F K (call periodically, QST=calling all stations).
P - Precisely align cursor to TS signal, to obtain good copy. The display center frequency will normally be at, or near, 1500. 
P - Once good copy of the Target Station is established, respond WX3F WX3F de KB3XYZ KB3XYZ, I HAVE IMAGE FOR RELAY TO N3XL K.
TS - RR KB3XYZ good copy, pls send your picture K.
P ? Send image. Terminate the transmission when image xfer is completed.
TS ? Brief acknowledgement, or request to resend, etc.


## Planning

File size limitations to achieve reasonable RF transmission times will vary, depending on the quality of your connection, propagation conditions, and equipment limitations. The nominal file size limit for Winlink and VarAC is 100 Kbytes, which is generally too large for RF transmissions. We are suggesting that you resize your images below 30 Kbytes for Vara FM and 15 Kbytes for HF. Make adjustments as necessary, if the transmission fails or takes too long.

You are encouraged to send images to as many Target Stations as possible. The HF and VHF Target Station lists will be provided to everyone on the MDC Winlink distribution list on 4 November 2025. Carefully observe the modes, frequencies, dates, and times of operation provided. VarAC HF users should use PSK Reporter, and Winlink Position maps to identify stations operating with flexible schedules.

A training video on Capturing VarAC HF Target Station Position Reports with detailed op schedule information is available at: https://drive.google.com/file/d/1J_NIODy1UTNTTzQETrboUOQQZyvH7DIX/view?usp=sharing

Winlink Packet P2P and Vara FM P2P users should consider the use of digipeating to extend their range beyond VHF line-of-sight. A VARA license is required to digipeat using Vara FM.
Ensure that your VARA, VarAC, Fldigi, and Winlink software programs are up-to-date.
VARA - https://rosmodem.wordpress.com
VarAC - https://www.varac-hamradio.com
Winlink - https://www.winlink.org/WinlinkExpress
Fldigi/Flmsg - https://sourceforge.net/projects/fldigi/files/

VarAC FM Stations should ensure that the following QSO tab check-boxes are left BLANK:
- Exchange Call ID and SNR after QSY from CF
- Consider entire frequency list as CF
- Auto QSY
- Allow Band skip QSY
- Send QSY Freq Offset

VarAC FM users should are encouraged to edit the VarAC FM Frequency Drop-down list to reflect the target stations/ digipeaters that you are able to use. For example:
