### **Contact and Channel de-duping**
At the request of Jerry from Connect Systems, the converter now supports de-duping of IDs (TalkGroups / Contacts) and channels. User is presented a dialog box to select the name of the Contact to use. Duplicate Contact Names are removed. Duplicate contact names in channels are replaced with the name the user selects. User is presented a dialog box of duplicate channel names and they must select which channel name to use. Duplicate channel names are removed.
